English Lessons

App is developed in Dart/Flutter

The main feature of application: 

 - The video player which is related to lecture

- Reference which is used text body of lecture

- Presentations of lecture are included in presentations section

- Admin can easily add their own glossaries, words, references,

- Tests which are related to lecture
